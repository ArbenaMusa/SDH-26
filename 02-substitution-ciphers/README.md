# Week 2 — Substitution Ciphers

Kjo javë mbulon algoritmet bazë të substitution ciphers dhe implementimin e tyre në Python.

## Temat e mbuluara
- Caesar Cipher
- Caesar Brute Force
- Frequency Analysis
- Atbash Cipher
- Vigenère Cipher
- One-Time Pad (OTP)

## Struktura
- `slides/` — slajdet e javës
- `examples/` — shembujt e implementuar në klasë
- `exercises/` — ushtrimet për studentët
- `solutions/` — zgjidhje ose udhëzime shtesë

## Si të ekzekutohen shembujt

python3 examples/caesar_cipher.py
